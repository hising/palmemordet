# Palmemordet

## Development`

Need to have webpack, babel, grunt, webpack-dev-server installed (globally). Built with Node 6.2+

```
$ cd palmemordet/docs
$ npm install
$ npm dev
$ grunt
```

## Deployment
```
$ cd palmemordet/docs
$ npm build
$ git add bundle.js
$ git commit -m "Added new build"
$ git push
```
